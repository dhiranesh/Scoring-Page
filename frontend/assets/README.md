# Assets Folder

This folder contains app assets like icons and splash screens.

## Required Assets

Replace the placeholder files with actual images:

- **icon.png** - App icon (1024x1024 px, PNG)
- **splash.png** - Splash screen (PNG, any size)
- **adaptive-icon.png** - Android adaptive icon (1024x1024 px, PNG)
- **favicon.png** - Web favicon (PNG, recommended 32x32 or 48x48 px)

## Generating Assets

You can use online tools or Expo's asset generation:
- https://appicon.co/ - Generate all sizes from one image
- https://expo.github.io/expo-icon-builder/ - Expo icon builder

Or use a design tool like Figma, Sketch, or Adobe XD.
